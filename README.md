# Figbook Downloader — EPUB & FB2

Пользовательский скрипт для сайта [Ficbook.net](https://ficbook.net), позволяющий скачивать фанфики в форматах **EPUB** и **FB2** для офлайн-чтения.

## Использование

1.  Откройте страницу любого фанфика на Ficbook.
2.  В шапке профиля (рядом с кнопками "Нравится" / "В сборник") появятся кнопки **EPUB** и **FB2**.
3.  Нажмите на нужный формат.
    *   *Примечание: Если вы находитесь на странице оглавления, скрипт предложит перейти на страницу "Все части одной лентой" для корректного захвата текста.*
4.  Файл будет сгенерирован и скачан автоматически.
